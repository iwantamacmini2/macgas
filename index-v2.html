1

Build your transaction

Create your Solana transaction as usual. No special SDK needed - use @solana/web3.js or any library you prefer.

2

Send to our API

POST your serialized transaction to our endpoint. We add a fee payer, sign it, and broadcast it to the network.

3

User transacts for free

Your user's wallet never needs SOL for gas. The transaction lands on-chain and everyone's happy.

Integration Example 
// Build your transaction (user signs)
const tx = new Transaction().add(instruction);
tx.feePayer = new PublicKey("FEE_PAYER_ADDRESS");
const signed = await wallet.signTransaction(tx);

// Send to macgas.xyz
const res = await fetch("https://macgas.xyz/sign_and_send", {
  method: "POST",
  headers: { "x-api-key": YOUR_API_KEY },
  body: JSON.stringify({ tx: signed.serialize().toString("base64") })
});
const { signature } = await res.json();
