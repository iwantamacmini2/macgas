1 Register Your Project

Head to the Dashboard to register your project and get an API key.

Choose your tier:

  • Pay-in-any-token ‚Äî Free for devs. Users pay ~$0.001/tx in USDC.
  • Sponsored ‚Äî You prepay, users transact free.

Save your API key ‚Äî you'll need it for API calls.

2 Fund Your Balance (Sponsored tier only)

If using the Sponsored tier, fund your balance via the Dashboard.

If using Pay-in-any-token, skip this step ‚Äî users pay per transaction.

3 Build Your Transaction

Create your transaction normally, but set the fee payer to our address:

import { Transaction, PublicKey } from '@solana/web3.js';

// Our fee payer address
const FEE_PAYER = new PublicKey(
  '3web11FBh1KuejjJTv8NjuaXGh6qgyXBFcx8khyJ8Wpv'
);

// Build transaction
const tx = new Transaction();
tx.add(yourInstruction);
tx.feePayer = FEE_PAYER;
tx.recentBlockhash = (await connection.getLatestBlockhash()).blockhash;

// User signs (proves intent, doesn't pay)
const signed = await wallet.signTransaction(tx);

Important: The user signs to authorize the action. They don't need any SOL.

Full Example

import { 
  Connection, Transaction, PublicKey, 
  SystemProgram, LAMPORTS_PER_SOL 
} from '@solana/web3.js';

const FEE_PAYER = new PublicKey('3web11FBh1KuejjJTv8NjuaXGh6qgyXBFcx8khyJ8Wpv');
const PROJECT_ID = 'your_project_id';

async function sendGasless(wallet, connection, instruction) {
  // Build transaction
  const tx = new Transaction().add(instruction);
  tx.feePayer = FEE_PAYER;
  tx.recentBlockhash = (await connection.getLatestBlockhash()).blockhash;
  
  // User signs
  const signed = await wallet.signTransaction(tx);
  
  // Send to gas station
  const res = await fetch('https://macgas.xyz/sign_and_send', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'X-Project-ID': PROJECT_ID
    },
    body: JSON.stringify({
      transaction: signed.serialize().toString('base64')
    })
  });
  
  return res.json();
}

// Usage

    fromPubkey: wallet.publicKey,
    toPubkey: recipientPubkey,
    lamports: 0.01 * LAMPORTS_PER_SOL
  })
);

console.log('Signature:', result.signature);

API Reference

POST /register

Register a gasless project (you prepay).

POST /payg/register

Register a pay-as-you-go project (users fund balance).

GET /balance/:projectId

Check project balance.

POST /sign_and_send

Submit a transaction. Requires X-Project-ID header.

GET /health

Service health check.

Devnet (Free Testing)

Test without real money on Solana devnet:

GET /devnet/info

Get devnet fee payer address and usage info.

POST /devnet/sign_and_send

Send gasless transactions on devnet. Free, no registration required.
